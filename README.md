# Genie Trigger Template using Cookiecutter

Genie [cookiecutter](https://cookiecutter.readthedocs.io/en/latest/) trigger
template to be used with the `pyats create trigger` command.


## Installation & Usage

```bash

# create new template
bash$ pyats create trigger
```
